Implements the "poke" array command which writes a value into an existing array variable. Uses the array helper logic provided by appArray to parse parameters at compile time and to perform the write at runtime.

CheckParameters(string[])

Validate tokens provided for a poke operation. Expects either 2 or 3 tokens after preprocessing (array name and row, optionally column). This method reports invalid parameter counts via a message box consistent with the rest of the project.
