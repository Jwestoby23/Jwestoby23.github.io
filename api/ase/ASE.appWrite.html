Implements the "write" command which outputs text to the canvas. Execution order:

  • Attempt to evaluate the parameter as an expression via the program.
  • If the evaluation fails, treat the parameter as a variable name and write its value.
  • Otherwise treat the parameter as a literal text string and write it.
