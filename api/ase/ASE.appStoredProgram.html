Class appStoredProgram

Assembly
ASE.dll

Application-specific BOOSE.StoredProgram implementation. Responsibilities:

  • Store and locate ASE variables.
  • Maintain a separate double backing for appReal variables and update both backing and textual value when UpdateVariable(string, double) is called.
  • Evaluate numeric expressions substituting real for real variables and BOOSE.Evaluation.Value for other variables before computing the result.

Returns the textual value for a variable used in outputs. For appReal variables this returns the underlying numeric backing formatted invariantly. For booleans returns "true"/"false" strings, otherwise returns the variable's textual value.

UpdateVariable(string, double)

Update a variable value. If the target variable is an appReal, its numeric backing (real) is updated and the textual/display value is set (using reflection fallback) so UI code that calls GetVarValue(string) sees the updated representation. Non-real variables are handled by the base implementation.
