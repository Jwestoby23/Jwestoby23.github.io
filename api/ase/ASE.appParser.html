Application-specific parser that converts a single line of ASE language into an BOOSE.ICommand. Extends the generic BOOSE.Parser to add ASE-specific preprocessing, comment handling and assignment resolution where the left-hand side references an existing variable.

ParseCommand(string)

Parse a single textual command line into a concrete BOOSE.ICommand. Behavior:

  • Lines beginning with a comment marker are converted to NoCommand.
  • Preprocesses the line (normalises '*' tokens).
  • Detects assignment syntax (e.g. "x = ...") and, if the left-hand side is an existing variable, rewrites the command so the variable's type-specific command (int/real) will be created and used.
  • Uses the injected BOOSE.CommandFactory to create the command, calls Set(StoredProgram, string), and then BOOSE.ICommand.Compile().
